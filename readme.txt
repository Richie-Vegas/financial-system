# Financial System

A basic command-line financial system written in Python to track income, expenses, and calculate balance.

## 📌 Features

- Add and track multiple income sources
- Add and categorize expenses
- Calculate total income, total expenses, and remaining balance

## 🧱 Project Structure

FinancialSystem/

## 🛠️ How to Run

1. Make sure Python is installed.
2. Navigate to the project folder in terminal.
3. Run the program:
